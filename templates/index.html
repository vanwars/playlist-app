Spotify Data Tester

Authentication

  • /spotify-login: Click this link to log in. Once you do, your token will be saved as a server session variable and you can query Spotify as if you're the current Spotify user.

Sample API Endpoints with Token Applied

From the perspective of the logged in user. If you're not logged in, should redirect you to /spotify-login

  • /api/track: Returns a hard-coded track
  • /api/recently-played: Current user's recently played tracks
  • /api/playlists: Current user's playlists
  • /api/user: Current user info
